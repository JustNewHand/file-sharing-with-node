***注意：首次使用时请先配置共享文件夹根目录及用户。同时保证运行本软件的机器的8080端口不要被占用。


=================================
共享文件夹根目录配置：

用记事本打开config.py，修改其中等号后方的路径为希望设置的共享文件夹根目录（要求为带盘符的绝对路径，如c:/test1/test2/），路径支持字母、数字、中文等。

***注意：路径中的斜杠要用正斜杠 ‘/’。


=================================
用户配置：

用记事本打开account文件夹下的user.db，其中第一行为name，password，从第二行开始为系统可用的用户名和密码，每对用户名和密码占一行，用户名和密码中间使用逗号间隔。

***注意：分隔的逗号请使用英文逗号‘,’。


=================================
系统启动方式：

共享文件夹根目录及用户配置无误后，双击startup.bat文件即可启动系统，然后在浏览器中输入本软件所在机器的ip地址加8080端口即可访问。

例如：192.168.5.123:8080



******另注意：修改任意配置项后均需重启本软件（关闭原先的startup.bat，再将其重新打开）以使配置生效。******
